Library

A P S College of Engineering has a spacious central library with an area of 586 Sq. mtrs. The library has fully automated EASY LIB ILMS software. Easylib Web Version License : Easy lib 6.4a Cloud Version and the library is automated since 2009. The faculty members and students of the institution utilize the library for continuous learning and to enhance their knowledge.

The library is well equipped with a reading capacity of 120 readers and also Wi-Fi enabled. The central library has various text Books, reference Books, e-books, e-journals, print Journals, Magazines, Project Reports, NPTEL video lessons related to Engineering and allied subjects.

Central library has a collection of fascinating books which includes Dictionaries, GATE and competitive examination books and videos that supports the content beyond syllabus.

The library provides four books per semester for each student and also made provision for students to refer previous semesters question papers of affiliating University for preparing their semester examinations. Central library has around 6356 number of Titles, 46101 Volumes, subscribed through VTU CONSORTIUM FOR E-RESOURCES & 30 No’s Hard copy Journals.
